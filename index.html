第一性原理重构：自动采集管道上线

围绕"如何让保险从业者以最低成本持续获取高价值资讯"这一本质目标，完成两项关键升级：

  • 数据失真修复（P1）：统一 ai_score 为 0-100 制；依据发布日期真实重算 days 统计；删除硬编码假 source_health；刷新信源更新时间；资讯按时间倒序重编 id。
  • 自动采集管道（P0）：新增零依赖 collect.py，支持 RSS 信源 + 收件箱 ingestion（inbox.json）双通道，自动评分、研究主题分类、Levenshtein 去重、增量合并，并依据真实数据重生 source_health。
  • 数据与前端解耦：data.json 地址经 <meta name="data-url"> 可配置；新增 GitHub Actions 每日定时运行采集并提交 data.json，使"发布资讯"成本趋近于零。
